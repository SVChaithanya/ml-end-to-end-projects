# Credit Risk Intelligence System

End-to-end machine learning system for predicting loan default risk and expected financial loss.

## Problem Statement
Financial institutions need accurate and explainable models to assess borrower risk and minimize losses.

## Solution Overview
This project builds a complete ML pipeline including:
- Data preprocessing and model training using LightGBM
- REST API deployment using FastAPI
- Interactive user interface using Streamlit
- Prediction logging using PostgreSQL

## Tech Stack
- Python
- LightGBM
- Scikit-learn
- FastAPI
- Streamlit
- PostgreSQL
